Jarvis GPT - Instrucciones de Uso (Offline)

1. **Abrir Jarvis:**
   - Descomprime el archivo ZIP si es necesario.
   - Haz doble clic en el archivo `index.html`. Se abrirá en tu navegador web predeterminado.

2. **Acceso:**
   - Se te pedirá una contraseña. Introduce `Luis850214#` para acceder a la interfaz de chat.

3. **Uso:**
   - Escribe tus preguntas o comandos en el campo de texto en la parte inferior.
   - Presiona Enter o el botón de enviar para obtener una respuesta de Jarvis.
   - Usa el botón del micrófono para dictar tus preguntas.

4. **Funciones:**
   - **Cambiar Tema:** Usa el botón del sol/luna para cambiar entre el modo claro y oscuro.
   - **Lectura de Voz:** Activa/desactiva la lectura automática de las respuestas de Jarvis con el botón del altavoz.
   - **Borrar Historial:** Haz clic en el icono de la papelera para limpiar la conversación actual.
   - **Cerrar Sesión:** El botón de salida te llevará de vuelta a la pantalla de la contraseña.
   - **Copiar/Descargar Código:** Cuando Jarvis genere código, aparecerán botones para copiarlo al portapapeles o descargarlo como un archivo .zip.

5. **Notas:**
   - Esta versión funciona con una conexión a Internet para usar el modelo de lenguaje avanzado.
   - Tu historial de conversación se guarda en tu navegador, por lo que si recargas la página, la conversación se restaurará. No se guarda en ningún servidor externo.
